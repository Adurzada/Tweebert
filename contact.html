The Query

The Query can be anything you would input in the search prompt in Twitter. It can be [Bitcoin], or [#BTC]
*[] indicate the query. You do NOT need to include them.
Basic Operators:
[Apple inc.] -> anything containing either Apple, inc. or both
['Apple inc.'] -> anything containing Apple inc.
[Apple OR APPL OR #APPL] -> anything containing one of these.
[Bitcoin -filter:retweets] -> filter retweets out
